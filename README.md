# elara.

Soft, chic storefront foundation for the elara. jewelry brand.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Local mock data only for Phase 1

## Development

```bash
npm run dev
npm run lint
npm run typecheck
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to preview the storefront.
